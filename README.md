## Requirements
- Minimum Java 11

## Run Development mode

Install the npm dependencies
```console
npm install
```

Compile the Typescript and Sass Assets
```console
npm run dev
```

Run Spring Boot
```console
./gradlew bootRun
```

Then simply visit the following link [http://localhost:4040/](http://localhost:4040/)

## Run Tests
```console
./gradlew test
```
